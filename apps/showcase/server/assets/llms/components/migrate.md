# migrate



## Overview

OpenVue is a fork of PrimeVue, and most existing PrimeVue v4 code already works against it with minimal changes. To make moving over effortless, we're building @openvue/migrate , a dedicated codemod that will automatically rewrite a project's imports and package references from PrimeVue to OpenVue. The package is published on npm, but it's still an early, active work in progress and isn't ready for use in real projects yet. Once it's ready for general use, this page will be updated with installation and usage instructions. In the meantime, you can track progress on npmjs.com/package/@openvue/migrate .
